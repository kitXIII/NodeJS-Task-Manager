# Task manager. NodeJS Web Application


## Requirements


* Node.js >= 10
* posgresql (as production env DB engine)
* sqlite (as test env DB engine)

## Local start

Run `make setup` to setup local environment

Run `make start` and open browser at http://localhost:3000
